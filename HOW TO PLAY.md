1.🎮 My Game

  📖 Description
  This project is a simple 2D Snake game created using Python and Pygame.  
  The game allows the player to move around the screen using keyboard controls. 
  It is designed for beginners who want to learn:
    - Game development
    - Python programming
    - Pygame basics
    - Movement systems
    - Game loops and events
  Through out my journet of creating this code i have understanded a lot of different topic about Python and IT in general,
2. 📌 Features
- Player movement
- Smooth controls
- Game window and background
- Basic game mechanics
- Beginner-friendly code

# 🕹 Controls

| Key | Action |
|------|---------|
| W | Move Up |
| A | Move Left |
| S | Move Down |
| D | Move Right |
